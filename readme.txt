=== AP Blocks ===
Plugin Name:       AP Blocks
Description:       Starter blocks package
Requires at least: 5.8
Requires PHP:      7.0
Version:           0.1.0
Author:            Anton Plauche


== Description ==

Library of blocks starter package. 

New blocks should be added in the blocks folder, blocks.json manifest, and finally the webpack.config.js entry points.

Run `npm run start` to watch all folders for changes and compile.

== Installation ==

Build with `npm run build`

Install as standard plugin or in the mu-plugins folder to prevent accidental deactivation.
